HW Week: 06/01 – 06/08
Focus: Linux, Python, Git

Description:

Create a mini applet that uses a Linux bash script to orchestrate several python files that
sit within nested directories below it. (If you did not attend class on 06/03/2026, then watch
the playback for step-by-step on how to set this structure up and execute it)

Requirements:

- The Linux bash script must use a dynamic variable to read in user input
- A conditional statement must be determined to which python script will be
executed
- There must be a minimum of 1 python script in each sub-directory (3 sub directories
total)
- Ensure that python scripts are named according to their basic functionality rather
than with generic names
- Ensure each python script uses a variable, an operator, and a conditional
- Ensure that both the bash script and the python scripts have appropriate logging
statements
